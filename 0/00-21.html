Catatan Tugas Kolaborasi KOMAS 2000

Pendahuluan

Berikut merupakan beberapa catatan perihal penggunaan sebuah sistem pendukung pada tugas kuliah Komputer dan Masyarakat. Untuk sementara berbentuk "dump" memo.

Memo

    Umum

  • semula menggunakan newsgroup "forum.iki.komas" ditemukan dua masalah dengan forum tersebut:
    • tidak ada fasilitas backup
    • threadnya sulit dikendalikan dan disort
  • mempertimbangkan menggunakan notes komas.nsf
  • sistemnya tidak user friendly, sangat membingungkan pengguna pemula. usul: mengurangi option awal dengan memperbanyak "default"
  • sistemnya sangat lambat, sering crash, dan tidak refresh secara otomatis.
  • penggunaan user komasXX tidak bisa ditraceback
  • penambahan user csXYY yang lebih universal
  • user bisa login tanpa password --> sehingga tidak terlacak siapa yang mengakses
  • sistem nyaris TIDAK digunakan untuk berkolaborasi
  • sistem digunakan untuk mengumumkan ujian. mengalami kesulitan karena banyak user yang lupa passwordnya
  • sistem digunakan untuk appointment mengambil hasil ujian.

    Administratif

  • sistemnya disknya bermasalah; mengapa tidak diinstall ulang oleh administrator?
  • diperlukan dukungan yang lebih baik
